## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/soilsnap.git
   ```
2. Install dependencies for both frontend and backend:
   ```
   cd SoilSnap
   cd backend
   npm install
   cd ../frontend
   npm install
   ```
3. Set up environment variables as needed.

4. Start the development servers:
   ```
   # In one terminal
   cd backend
   npm run dev

   # In another terminal
   cd frontend
   npm start
   ```
